# Paradise Nursery Shopping Application

## Project Overview
Paradise Nursery is a React + Redux front-end shopping application that allows users to browse a variety of houseplants, add them to a shopping cart, and manage their cart before checkout.

## Features
- Landing page with background image, company info, and Get Started button
- Product listing page with 18+ houseplants organized into 3 categories
- Shopping cart with quantity management and total cost calculation
- Redux state management for cart operations
- Deployed via GitHub Pages

## Tech Stack
- React 18
- Redux Toolkit
- Vite
- CSS3
